Placeholder for PWA icons. Generate real icons using:

- `icon-32x32.png` - Favicon
- `icon-180x180.png` - Apple touch icon
- `icon-192x192.png` - Android icon
- `icon-512x512.png` - PWA splash screen

## Quick generation:
Create a 512x512 PNG icon with your logo and use an online generator to create all sizes.
